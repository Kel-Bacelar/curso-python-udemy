# Curso de Python 3 - Do Básico ao Avançado (Portfólio de Projetos) 🚀

Este repositório reúne os códigos, projetos práticos e soluções arquiteturais desenvolvidos por mim durante o curso de Python 3 mais completo do mercado. O objetivo deste espaço é demonstrar a evolução das minhas habilidades técnicas, desde a lógica fundamental até o desenvolvimento de aplicações prontas para produção.

---

## 🛠️ Stack Tecnológica & Competências Desenvolvidas

Abaixo estão as principais tecnologias, frameworks e conceitos de engenharia de software consolidados ao longo de **mais de 141 horas de código e 751 aulas**:

### 🎯 Core Language & Arquitetura
*   **Python 3 Avançado:** Domínio absoluto de sintaxe, Programação Procedural, Funcional e Orientada a Objetos (POO) no padrão Pythônico.
*   **Padrões de Projeto (Design Patterns):** Implementação prática dos padrões da GoF (Gang of Four) para criação de código limpo, reutilizável e de fácil manutenção.
*   **Testes Automatizados & Qualidade:** Desenvolvimento orientado a testes (**TDD básico**) utilizando ferramentas como `unittest` e `pytest`.

### 🌐 Desenvolvimento Web & APIs
*   **Framework Django:** Construção de aplicações web completas, sites dinâmicos e APIs robustas.
*   **Bases de Dados Relacionais:** Modelagem de dados, relações entre tabelas e consultas complexas utilizando **SQLite3, MySQL e MariaDB**.

### 💻 Engenharia de Software & Automação
*   **Interfaces Gráficas (GUI):** Desenvolvimento de aplicações desktop modernas utilizando **PySide6 (Qt 6)**.
*   **Automação & Web Scraping:** Criação de scripts para raspagem de dados e automação de processos com **Selenium**.
*   **Expressões Regulares (RegEx):** Manipulação avançada e validação complexa de strings.

### ☁️ Infraestrutura & DevOps (Extra)
*   **Ambiente Unix:** Domínio de comandos de terminal em ambientes Linux, macOS e Windows WSL2.
*   **Deploy & Cloud:** Criação, configuração e deploy de servidores Linux no **Google Cloud Platform (GCP)** com gerenciamento de chaves SSH.
*   **Front-end Basics:** Integração de interfaces web utilizando HTML5 e CSS3.
